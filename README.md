# FlexFlow ⚡

> **Stay consistent, become the athlete.**

FlexFlow is a behavior-driven fitness consistency platform built with Flutter and Firebase. Unlike traditional workout logs, FlexFlow focuses on the psychological and social factors that sustain long-term habits.

![Premium UI Mockup](https://raw.githubusercontent.com/parishrivastava632-sys/Priya/main/flexflow_banner.png) *(Placeholder for your future screenshot)*

## 🧠 Core Features

### 1. Motivation Engine
Uses **Identity-based Nudges** to reinforce your fitness habit.
- **On Track**: Boosts your identity as an athlete.
- **Slipping**: Acts as a "Momentum Guard" to prevent drop-offs.
- **Off Track**: Provides a "Fresh Start" mindset.

### 2. Adaptive Intelligence
Calculates a rolling **7-day Consistency Score**. The app dynamically pivots workout intensity and suggestions based on your real-world behavior.

### 3. Gamification Layer
- **XP & Levels**: Earn points for every consistent day.
- **Achievement Badges**: Unlock badges like "Consistency King" and "7-Day Comeback".

### 4. Community & Connectivity
- **Social Leaderboard**: Rank among peers based on consistency, not just raw power.
- **Wearable Sync**: Integration (Mock) with Google Fit and Apple Health.

## 🛠️ Tech Stack
- **Frontend**: Flutter (Dart)
- **State Management**: Provider
- **Backend**: Firebase (Auth & Firestore)
- **Aesthetic**: Glassmorphism & Cyber-Fitness Theme

## 🚀 Getting Started

1. **Clone the repo:**
   ```bash
   git clone YOUR_REPO_URL
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase:**
   Follow the [FlutterFire Guide](https://firebase.flutter.dev/docs/overview/) to link your project.

4. **Run the app:**
   ```bash
   flutter run
   ```
